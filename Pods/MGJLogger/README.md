## 关于 MGJLogger

MGJLogger 是针对 CocoaLumberjack 做的定制，相比NSLog速度更快，支持不同级别的日志输出以及输出日志到控制台和文件，目前最新版本0.1.0

## 使用 MGJLogger

pod 'MGJLogger', '= 0.1.0'

下载demo工程，里面有详细的使用方式
